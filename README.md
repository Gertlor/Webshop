#Webshop
In dit project wordt er een webshop gemaakt in Angular 4 en Dropwizard

##Hsleiden
Dit project is in opdracht van Hsleiden. De webshop is niet echt.
Website URL: gerlobes.nl

##Uri's

    POST    /api/account/register

    GET     /api/product/all

    GET     /api/product/{ids}
